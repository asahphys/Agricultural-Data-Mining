# 🌾 Agricultural Data Mining Mini Project

## 📌 Overview
This project aims to analyze factors affecting crop yield using data preprocessing, data mining techniques, and exploratory data analysis (EDA).

## 📊 Dataset
The dataset consists of the following variables:
- Temperature (suhu)
- Humidity (kelembaban)
- Rainfall (curah_hujan)
- Land Area (luas_lahan)
- Crop Yield (hasil_panen)
- Yield Category (kategori_hasil)

## ⚙️ Methods
- Data Preprocessing (handling missing values & normalization)
- Classification using Decision Tree
- Model Evaluation (accuracy & confusion matrix)
- Exploratory Data Analysis (EDA)

## 📈 Results
- Model achieved **100% accuracy**
- Rainfall shows a strong positive relationship with crop yield
- Temperature, humidity, and land area have low correlation with yield
- Potential indication of overfitting due to perfect accuracy

## 📊 Visualization
- Data visualization using Matplotlib & Seaborn
- Additional dashboard created using Tableau

## 🚀 Key Insights
- Rainfall is the most influential factor affecting crop yield
- Other variables show minimal impact
- Dataset patterns are relatively simple, leading to high model performance

## ⚠️ Limitations
- Small dataset size (~100 samples)
- Limited features (no soil quality, fertilizer, etc.)
- Possible overfitting in the classification model
